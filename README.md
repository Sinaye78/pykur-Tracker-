# Pykur Tracker

Pykur Tracker est un outil de suivi de progression pour le donjon **Docteur Morose** sur DOFUS.

Le tracker permet de suivre :

- les donjons effectués/restants
- les monstres vaincus
- la prospection gagnée
- le temps de farm
- les statistiques de progression
- la progression globale avec barre animée
- les sessions de jeu et le chronomètre

## Fonctionnement

Le projet fonctionne entièrement en :

- HTML
- CSS
- JavaScript

Aucun logiciel à installer.

Il suffit d’ouvrir :

`index.html`

dans un navigateur.

## Fonctionnalités

- Sauvegarde automatique locale
- Barre de progression dynamique
- Notifications visuelles
- Effets sonores
- Chronomètre intégré
- Statistiques détaillées
- Temps estimé restant
- Temps moyen par donjon personnalisable
- Désactivation des sons
- Désactivation des animations
- Apparition d’AuraPykur à 100 %

## Sécurité

Le projet :

- ne contient aucun `.exe`
- ne contient aucun `.bat`
- n’utilise aucun serveur
- n’envoie aucune donnée
